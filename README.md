## How to Use This System

### If you're SELLING:
1. Start at `01-Strategy/Business-Foundation.md` — know your ICP and verticals
2. Follow `02-Sales/Discovery-Process.md` — run every call the same way
3. Use the Research Agent (`06-Internal-Agents/Agent-01-Research-Agent.md`) to prep for calls
4. Deliver the audit using `08-Templates/Automation-Audit-Template.md`
5. Generate proposals with the Proposal Agent (`06-Internal-Agents/Agent-02-Proposal-Agent.md`)
6. Reference pricing from `07-Service-Packages/` for the relevant vertical

### If you're DELIVERING:
1. Follow `05-SOPs/SOP-01-New-Client-Engagement.md` — stage by stage
2. Use `03-Delivery/Client-Onboarding-Checklist.md` to kick off
3. Follow `03-Delivery/Delivery-Workflow.md` for each phase
4. Build with standards from `05-SOPs/SOP-02-Implementation-Standards.md`
5. Pull prompts from `04-Prompt-Libraries/Client-Facing-Prompts.md`
6. QA everything with `05-SOPs/SOP-03-Quality-Assurance.md`

### If you're REPORTING:
1. Use the Reporting Agent (`06-Internal-Agents/Agent-03-Reporting-Agent.md`)
2. Generate monthly client reports and weekly internal updates

### If you're TRAINING someone new:
1. Have them read this README first
2. Then `01-Strategy/Business-Foundation.md`
3. Then the 3 SOPs in order
4. Then shadow a live engagement
